# TikTok Live Translator — PWA

Real-time mic translation for TikTok live streams.
Free, no API key, works on Android and iPhone.

---

## 🚀 How to host for FREE on GitHub Pages

### Step 1 — Create a GitHub account
Go to https://github.com and sign up (free).

### Step 2 — Create a new repository
1. Click the "+" icon → "New repository"
2. Name it: `tiktok-translator`
3. Set it to **Public**
4. Click "Create repository"

### Step 3 — Upload these files
1. Click "uploading an existing file"
2. Drag and drop ALL files from this folder:
   - index.html
   - app.js
   - manifest.json
   - sw.js
   - icon-192.png
   - icon-512.png
3. Click "Commit changes"

### Step 4 — Enable GitHub Pages
1. Go to your repo → Settings → Pages
2. Under "Source" select: **Deploy from a branch**
3. Branch: **main** / Folder: **/ (root)**
4. Click Save

### Step 5 — Your URL is ready!
After ~1 minute your app will be live at:
👉 `https://YOUR-USERNAME.github.io/tiktok-translator`

Share this URL with anyone — it works on any phone browser!

## 📱 How to use on your phone

### Android (best experience)
1. Open the URL in **Chrome**
2. Tap the 3-dot menu → "Add to Home Screen"
3. Open TikTok Live in another tab or app
4. Use **Split Screen**: hold the recent apps button,
   tap the TikTok window, then select Chrome for the bottom
5. The translator subtitles show in real time at the bottom

### iPhone
1. Open the URL in **Safari**
2. Tap the Share icon → "Add to Home Screen"
3. Open TikTok, play the stream with sound
4. Swipe up and switch to the Translator app
5. Tap Start — it listens through your mic
6. Subtitles appear and stay visible as you switch back

### 💡 Pro tips
- **Turn up your phone volume** so the mic picks up the stream clearly
- Use **XL subtitle size** for easiest reading
- Portuguese → English is selected by default
- Tap **Save** to download a transcript after the stream
- The app works **offline** after first load (cached)
